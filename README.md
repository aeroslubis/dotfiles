# Dotfiles

My personal dotfile for Archlinux 2019, based of another person dotfiles

## Requirement
- Git
- GNU Stow
- i3wm
- polybar
- i3blocks
- rofi
- termite
- mpd
- mpv
- ncmpcpp
- sxiv
- youtube-dl
- dunst

## Installation


```bash
git clone aeroslubis/dotfiles.git ~/dotfiles
stow -v -t ~/ dotfiles
```
